Feito por Gabriel Alan M. Gonçalves

    As atualizações para realização da Atividade 13 estão no menu Marcações -> Registrar Marcação de Funcionário
  • Apresentar uma lista de dados com a diretiva estrutural ngFor: Lista de funcionários renderizada em componente de busca de funcionários por nome;
  • Usar a diretiva estrutural ngIf para mostrar ou esconder elementos: O componente de busca de funcionários esconde ou mostra em uma listagem os funcionários que possuam o texto digitado em seu nome;
  • Formatar a apresentação de dados com Pipes: Ao registrar o funcionário, é exibido um modal que mostra a data e hora da operação, formatados com o pipe date;
  • Fazer requisições a API com tratamento da resposta com Observables: A busca dos funcionários foi migrada de uma busca em WebStorage para uma chamada ao JSON Server, utilizando Observable.
