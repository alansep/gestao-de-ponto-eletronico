Feito por Gabriel Alan M. Gonçalves

  • Mapear componentes à rotas no módulo de rotas.
  • Criar navegação entre páginas por meio de rotas.
  • Passar dados via hierarquia de componentes, ou seja, usando @Input ou @Output. OBS: o @Output está sendo usado no input de busca.
  • Passar dados entre componentes que representam diferentes telas via parâmetros de rotas. OBS: Este caso se trata ao clicar no nome de um funcionário na busca de funcionários.
